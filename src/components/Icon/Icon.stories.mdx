### Icon

The component renders an `svg` element with the link to the corresponding symbol
inside the svg-sprite.

```js
<svg>
  <use href="#icon-name" />
</svg>
```

SVG-sprite with icons is injected into the document body in
`./storybook/preview-body.html` for simplicity.

Icon source files can be found in `src/icons`.

#### Arguments

The component is passed two arguments:

- `size`, defines the size of an icon, 20x20 px for `s` (default) and 24x24 px for `l`
- `name` (required), corresponds to a symbol in the svg-sprite
