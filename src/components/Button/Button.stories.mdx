### Button

The component renders a `button` element with icon and label inside.

#### Arguments

- `type`, defines the button theme `primary`, `secondary`, `flat` and `destructive`

<Canvas>
  <Story name="primary" args={{ type: 'primary', label: 'Primary' }}>
    {Template.bind({})}
  </Story>
  <Story name="secondary" args={{ type: 'secondary', label: 'Secondary' }}>
    {Template.bind({})}
  </Story>
  <Story name="flat" args={{ type: 'flat', label: 'Flat' }}>
    {Template.bind({})}
  </Story>
  <Story
    name="destructive"
    args={{ type: 'destructive', label: 'Destructive' }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

- `size`, defines the size of the button `s` (default), `l`

<Canvas>
  <Story name="small" args={{ type: 'secondary', label: 'Small', size: 's' }}>
    {Template.bind({})}
  </Story>
  <Story name="large" args={{ type: 'secondary', label: 'Large', size: 'l' }}>
    {Template.bind({})}
  </Story>
</Canvas>

- `label`, defines the label on the button

- `iconName`, icon to use inside the button (see [Icon component](?path=/docs/docs-icon--info)).
  Does not render icon if `undefined`

- `iconPosition`, icon position `left`, `right` and `only`. Does not render icon if `undefined`

<Canvas>
  <Story

  <Story
    name="right"

    {Template.bind({})}
  </Story>
</Canvas>

- `disabled`, disable the button

<Canvas>
  <Story
    name="disabled"
    args={{ type: 'secondary', label: 'Disabled', disabled: true }}

    {Template.bind({})}
  </Story>

- `loading`, indicates loading state. Works for `primary` and `secondary` buttons only, otherwise ignored

<Canvas>
  <Story

  >

  <Story
    name="secondary-loading"

  >
    {Template.bind({})}
  </Story>
</Canvas>

- `onClick` callback, ignored if `undefined`
